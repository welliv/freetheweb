# FreeTheWeb — Hackathon Project

## The Vision
FreeTheWeb.net lets anyone build a complete website using natural language. An AI agent browses component libraries, wires Stripe payments, and deploys to Cloudflare — so you own your code and host for free.

## How It Works
1. User types: "Build me a luxury hat store"
2. Agent browses component library → assembles HTML pages with Tailwind CSS
3. Agent creates Stripe products, prices, and checkout links
4. Agent pushes code to a private GitHub repository under the user's account
5. Agent deploys to Cloudflare Pages (free tier: unlimited bandwidth, commercial use)
6. User owns the code. Zero hosting fees. Full control.

## The Problem
- Webflow charges $74/month for e-commerce
- Vercel bans commercial use on the free tier
- Manual setup takes 10-20 hours per project
- Your code is locked inside someone else's platform
- You don't truly own your digital presence

## The Solution
- Agent does everything: browses components, wires payments, deploys
- Cloudflare Pages: unlimited bandwidth, free forever
- GitHub: your code, your repo, export anytime
- Stripe: real payments, subscriptions, checkout
- Chatwoot: live customer support widget

## Tech Stack
- Python assembly engine (config → complete HTML)
- 8 premium Tailwind CSS components
- Stripe MCP integration (products, prices, payment links)
- Cloudflare Pages API deployment
- GitHub repo automation
- Chatwoot live chat integration
