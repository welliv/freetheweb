# FreeTheWeb — Build Websites with Natural Language

## Overview
An AI agent that takes natural language input, assembles websites from components, wires Stripe payments, and deploys to Cloudflare — giving users full code ownership and zero hosting fees.

## Learning Objectives
The viewer will understand:
1. The problem with current website builders (lock-in, high costs, no ownership)
2. How the AI agent orchestrates the entire build process
3. Why FreeTheWeb is the future of web development
